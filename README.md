# Modern Under Development Landing Page

A sleek, beautiful, modern "Under Development" landing page built with pure HTML, CSS, and plain JavaScript.

## Features

- **Glassmorphism Design:** Beautiful translucent cards with blurred backgrounds creating depth.
- **Ambient Animations:** Continuously moving glowing shapes in the background for a modern, premium feel.
- **Interactive 3D Hover Effects:** The main card follows your mouse smoothly on desktop to create a deep parallax effect.
- **Gradient Typography:** Stunning gradient text styles that animate naturally.
- **Responsive Design:** Looks great on desktop, tablet, and mobile displays.

## Deployment on Vercel

This project is ready to be deployed instantly on Vercel without any build configuration, as it consists entirely of static HTML, CSS, and JS files.

1. Push this repository to GitHub.
2. Log into [Vercel](https://vercel.com/) and click **Add New** > **Project**.
3. Import this GitHub repository.
4. Leave the "Framework Preset" as `Other` (or Vercel will auto-detect it as static).
5. Leave the "Build Command" and "Output Directory" blank.
6. Click **Deploy**.

## Local Development

Simply open `index.html` in your web browser, or serve it using an extension like Visual Studio Code's "Live Server" for hot reloading. No `npm install` or build steps required.

## Tech Stack

- HTML5
- CSS3 (Variables, Animations, Flexbox)
- Vanilla JavaScript (Event Listeners, DOM manipulation)
